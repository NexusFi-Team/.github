## 📋 프로젝트 개요
NexusFi는 현대인의 복잡한 금융 생활을 하나로 연결하는 **마이데이터 기반의 자산 관리 솔루션**입니다. 여러 금융 기관에 흩어져 있는 자산 정보를 실시간으로 통합하고, 이를 직관적인 대시보드 형태로 제공하여 사용자가 능동적으로 자신의 경제 상황을 관리할 수 있도록 돕는 웹 기반 애플리케이션입니다. 

이 프로젝트는 백엔드 개발자와 프론트엔드 개발자 2명이 각자의 전문성을 바탕으로 실무적인 문제 해결 과정을 기록하고 성장하기 위해 시작되었습니다.

## 🔍 기획 배경 및 목적
### 1. 금융 데이터의 파편화
현재 많은 사용자가 여러 개의 은행 계좌, 카드, 증권 계좌를 보유하고 있으나, 이를 통합적으로 관리하기 위해서는 각 앱에 개별적으로 접속해야 하는 번거로움이 있습니다.
### 2. 가계부 작성의 한계
기존 수동 가계부는 입력의 불편함으로 인해 지속성이 떨어지며, 자동 가계부 역시 단순한 내역 나열에 그쳐 실질적인 자산 흐름 파악에는 한계가 있습니다.
### 3. 데이터 기반의 의사결정 필요성
단순한 지출 기록을 넘어, 축적된 데이터를 바탕으로 미래 자산을 예측하고 소비 습관을 개선할 수 있는 '데이터 드리븐(Data-Driven)' 금융 비서의 필요성이 증대되고 있습니다.

## ✨ 핵심 서비스 가치
- **Connectivity (연결성)**: 마이데이터 표준 API를 통한 전 금융권 데이터의 유기적인 통합.
- **Clarity (투명성)**: 복잡한 수치를 시각화하여 자산 흐름을 한눈에 투명하게 파악.
- **Intelligence (지능화)**: 사용자의 소비 패턴을 학습하여 개인화된 금융 인사이트 제공.

## 💡 상세 화면 및 기능 기획

### 1. 인증 및 보안 (Login)
- **간편 로그인**: OAuth 2.0 및 JWT 기반의 안전하고 빠른 인증 시스템을 제공합니다.
- **데이터 연동 보안**: 금융 데이터 조회 시 사용자의 민감 정보를 암호화하여 처리하며, 보안 가이드라인을 엄격히 준수합니다.

### 2. 금융 현황 간편 조회 (Main Dashboard)
- **카테고리별 자산 비중**: 계좌, 카드, 대출 데이터를 도넛 그래프 형태로 시각화하여 자산 구성비를 한눈에 보여줍니다.
- **소비 요약 정보**: 이번 달 총 소비액과 이번 연도 누적 소비액을 카드 리스트 형태로 제공하여 지출 규모를 직관적으로 파악할 수 있게 합니다.

### 3. 심층 자산 분석 (Asset Analysis)
- **금융사별 자산 현황**: 각 은행 및 증권사별 자산 리스트를 상세히 제공합니다.
- **자산 포트폴리오**: 전체 자산을 도넛 그래프로 표시하여 유동 자산과 비유동 자산의 비율을 분석합니다.

### 4. 소비 현황 및 상세 조회 (Consumption Status)
- **소비 캘린더**: 캘린더 형태의 UI를 통해 날짜별 소비 금액을 표시하여 지출 패턴을 일자별로 추적합니다.
- **카테고리 분석**: 상세 화면에서 카테고리별 막대 그래프를 통해 어떤 분야에 지출이 집중되었는지 분석합니다.
- **태그별 지출 관리**: 소비 항목별 태그(식비, 주거, 취미 등)를 활용하여 금액 리스트를 상세하게 조회할 수 있습니다.

## ⚙️ 데이터 및 시스템 전략 (Backend Insight)

### 1. 실시간 데이터 동기화 및 배치 집계
- **Spring Batch**를 활용하여 수만 건의 거래 데이터를 주기적으로 수집하고, 카테고리별/일별/월별로 통계 데이터를 미리 생성(Pre-aggregation)하여 대시보드 조회 성능을 극대화합니다.
- **Redis**를 활용해 자주 조회되는 자산 현황 데이터를 캐싱하여 응답 시간을 단축합니다.

### 2. 지능형 알림 및 분석 기능 (추가 기획)
- **지출 임계치 알림**: 사용자가 설정한 예산을 초과할 경우 실시간 알림을 발송하여 과소비를 방지합니다.
- **고정 지출 탐지**: 반복되는 결제 패턴을 분석하여 구독 서비스나 공과금 등의 고정 지출을 자동으로 식별하고 관리해 드립니다.
- **비교 분석 리포트**: 지난달 대비 소비 증감률을 분석하여 사용자에게 정기적인 리포트를 제공합니다.

## 👥 타겟 사용자 및 시장 분석
- **MZ세대 사회초년생**: 체계적인 자산 형성의 기틀을 마련하고자 하는 초기 자산 관리 계층.
- **다중 계좌 보유자**: 여러 금융 기관을 이용하며 통합 관리의 효율성을 중시하는 스마트 컨슈머.
- **데이터 중심의 자산가**: 자신의 소비 기록과 자산 흐름을 시각적으로 분석하고 관리하기를 즐기는 사용자.

## 📈 비즈니스 컨설팅 제언
- **B2C 확장성**: 향후 데이터 분석 결과를 바탕으로 개인 맞춤형 예적금 및 카드 상품 추천 서비스로 확장 가능합니다.
- **데이터 비즈니스**: 비식별화된 지출 트렌드 데이터를 분석하여 시장의 소비 흐름을 파악하는 리포트 서비스를 고려할 수 있습니다.

## 🗺️ 서비스 로드맵
- **Phase 1 (MVP)**: 마이데이터 연동 기반 계좌/카드 통합 조회 및 기초 자산 현황 대시보드 출시.
- **Phase 2 (Growth)**: 소비 패턴 상세 분석 리포트 및 지출 목표 설정/알림 기능 강화.
- **Phase 3 (Optimization)**: AI 기반 자산 예측 모델 적용 및 개인 맞춤형 금융 상품 큐레이션 서비스 고도화.
